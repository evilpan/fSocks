fSocks
===

A hackable socks5 proxy.

**CURRENTLY WORKING IN PROGRESS**

# RUN

## on your local machine

```
python3 fclient.py -c config.json
```

## beyond the wall

```
python3 fserver.py -c config.json
```

# TEST

python3 -m unittest discover -s tests
